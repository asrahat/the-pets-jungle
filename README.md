# 🐾 The Pets Jungle

## 📌 Purpose

The Pets Jungle is a full-stack pet adoption platform where users can explore pets, submit adoption requests, and manage pet listings through a modern dashboard system.

---

## 🌐 Live URL

https://the-pets-jungle.vercel.app

---

## ✨ Features

* 🔐 Authentication with Better Auth
* 🐶 Google Login & Email/Password Login
* 🔎 Search & Filter Pets by Species
* ❤️ Send & Manage Adoption Requests
* 📋 Dashboard for Managing Pet Listings
* ✏️ Update & Delete Pet Listings
* 📱 Fully Responsive Design
